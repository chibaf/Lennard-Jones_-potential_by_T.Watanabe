# Lennard-Jones-potential_by_T.Watanabe
MD simulation for Ar with Lennard-Jones potential

## code
LJc.c : MD with LJ potential by T. Watanabe

## usage
gcc -O3 LJc.c

./a.out

outputs are LJ.dat, LJ.xyz
